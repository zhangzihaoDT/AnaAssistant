# LS8 上市后前3日锁单预测模型 (基于全周期推演倒推)

## 1. 建模逻辑优化
在原始预测中，我们仅使用“盲订客户占比”来推算上市后前 3 日的爆发。但这样忽略了预售整个周期的真实客流积淀。
现在，我们将逻辑调整为更严密的**“由全周期倒推极短期”**模型：
1. 先利用历史车型的 `预售前3日占比` 和 `预售前7日占比`，结合 LS8 真实数据推算出 **LS8 整个预售周期的累计留存小订总数**。
2. 将总小订乘以 30 日整体转化率，得到 **LS8 上市后 30 日的总锁单预测**。
3. 提取历史车型中 **上市后前3日锁单数占30日总锁单数的比例**。
4. 将该比例乘以预测的 30 日总锁单数，得出最终的 **LS8 上市后前3日锁单爆发预测**。

## 3. 一元线性回归模型 (30日总锁单 -> 上市前3日锁单)
直接拟合历史车型的 `30日总锁单数` (X) 与 `上市后前3日锁单数` (Y)。
- **回归方程**：`上市前3日锁单 = 0.37 × 30日总锁单数 + 510.94`
- **回归预测值**：输入前面 6 种对标推演出的 LS8 30日总锁单的平均值 (10,338 单)，得出上市后前 3 日锁单预测为：**4,302 单**。

## 4. 业务结论
经过这一轮从“宏观（全周期蓄水）”向“微观（前3天爆发）”的闭环修正后，预测数据变得更加符合现实商业逻辑。
LS8 尽管前3天盲订极高，但结合其前7天的长尾表现，它更像是一个“后劲充足但首波爆发并不极端畸形”的健康车系。
- 如果它表现出爆款特质（类似 CM1、CM2），它前3天可以锁单 **5,500 ~ 6,500 单**。
- 如果表现平庸，前3天锁单大约在 **3,800 ~ 4,300 单** 之间。
