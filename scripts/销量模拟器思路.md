# 二、你的核心假设

你提到一个关键假设：

> 在不发生重大业务变化的情况下，未来是当前表现的延长线。

数学上其实就是：

**趋势延续假设**

可以写成：

```
未来销量 ≈ 当前业务结构 × 当前效率
```

例如：

```
销量
=
线索量
× 锁单率
```

# 四、你的区间预测逻辑

你提出的 **MAX / MIN 预测**非常合理。

例如：

## 下限预测（MIN）

假设：

```
未来业务维持最近平均水平
```

例如：

```
最近30天
平均线索 = 3000
平均锁单率 = 4%
```

预测：

```
销量 = 3000 × 4% = 120
```

---

## 上限预测（MAX）

假设：

```
未来达到历史活动期水平
```

例如：

```
活动期锁单率 = 6%
```

预测：

```
销量 = 3000 × 6% = 180
```

# 五、预测区间

于是预测结果变成：

```
MIN = 120
MAX = 180
```

也就是：

```
120 ≤ 未来销量 ≤ 180
```

这种其实就是一种 **置信区间预测**。

# 七、统计表达

如果用统计语言描述，其实就是：

当前指标在历史分布中的位置。

例如：

使用 **Z-score**：

含义：

- (x) = 当前指标
- (\mu) = 历史平均
- (\sigma) = 历史波动

如果 Z 很高：
