# 三、核心函数（可直接复制）

## ✅ 1. 计算趋势值

```python
def calc_trend_delta(series, head_n=3, tail_n=3):
    """
    series: List[float]，例如 [0.5, 0.6, 0.7, ...]
    返回：delta（末端 - 起始）
    """
    if len(series) < head_n + tail_n:
        return None  # 数据不足

    head = series[:head_n]
    tail = series[-tail_n:]

    head_mean = sum(head) / len(head)
    tail_mean = sum(tail) / len(tail)

    return tail_mean - head_mean
```

---

## ✅ 2. 趋势分类（方向）

```python
def classify_trend(delta, threshold=0.05):
    """
    threshold：控制敏感度（建议 0.05）
    """
    if delta is None:
        return "unknown"

    if delta > threshold:
        return "up"
    elif delta < -threshold:
        return "down"
    else:
        return "flat"
```

---

## ✅ 3. 封装成统一输出

```python
def build_trend(series):
    delta = calc_trend_delta(series)

    return {
        "direction": classify_trend(delta),
        "strength": round(delta, 3) if delta is not None else None
    }
```

## Step 1：定义 Structure 因子

### 示例：

| 因子             | 构成                              | 业务意义     |
| ---------------- | --------------------------------- | ------------ |
| Volume           | 锁单数、下发线索数                | 流量规模     |
| Conversion       | 下发线索30日锁单率、当日锁单率    | 转化水平     |

## Step 2：计算 Structure Score

- 对每个因子做**分位化**
- 然后归一化到 0–1

```python
# 示例
structure_factors = {
    "Volume": ["锁单数", "下发线索数"],
    "Conversion": ["下发线索30日锁单率", "下发线索7日锁单率"],
    "LeadQuality": ["门店线索占比", "直播线索占比", "平台线索占比"]
}

def calc_structure_score(row, history):
    scores = {}
    for factor, fields in structure_factors.items():
        vals = [percentile(row[f], history[f].values) for f in fields]
        # 简单平均
        scores[factor] = sum(vals) / len(vals)
    return scores
```

## Step 2：定义相似性指标

```python
import numpy as np

def find_peer_days(target_vector, history_vectors, top_k=3):
    distances = []
    for date, vector in history_vectors.items():
        dist = np.linalg.norm(np.array(target_vector) - np.array(vector))
        distances.append((date, dist))
    distances.sort(key=lambda x: x[1])
    return distances[:top_k]
```
